# Java Core Assessment – Task Management System

## Project Overview

This project is a **Java-based Task Management System** that demonstrates core Java concepts such as:

* Custom Linked List implementation
* Multithreading
* Object-Oriented Programming (OOP)

The system allows managing tasks and processing them using multiple threads.

---

## Features

* Custom implementation of Linked List (without using Java Collections)
* Multithreaded task execution
* Structured package-based project design

---

## Technologies Used

* Java
* IntelliJ IDEA

## Execution of the Project

1. Clone the repository:

```
git clone https://github.com/sailesh-1725/Java-Core-Assessment.git
```

2. Open the project in IntelliJ IDEA

3. Navigate to:

```
src/com/javaCoreAssessment/Main.java
```

4. Run the `Main` class
